### Contributing to CosmicFirewall

Thank you for your interest. This project is a conceptual, theoretical work and welcomes careful, constructive contributions.

Guidelines:
- Use issues to request or discuss substantive changes. Tag your issue with one of: `reference`, `math`, `simulation`, `review`, `discussion`.
- For code or derivations, open a feature branch, add tests or a short reproducible script, and open a PR. Keep PRs focused.
- No hazardous experimental protocols will be accepted. Any contribution that includes dangerous procedures will be paused and reviewed.
- Respect the Code of Conduct and sign contributors' agreements where asked.

Reviewers should use `REVIEW_GUIDELINES.md` (included) to structure comments.
